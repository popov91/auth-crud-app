# Тестовое задание для PHP-разработчика.

Тестовое задание:
Необходимо написать на Laravel мини проект - CRUD новостей с категориями, c регистрацией и авторизацией

Функционал:
- читать все новости могут залогиненные и не залогиненные пользователи, как все так и по категории
- создавать только залогиненные
- редактирование новости может только автор новости
- удалять новость может только автор и супер-админ, также супер-админ может удалять рубрику, если удаляется рубрика - за ней все новости в этой категории (с удалением сделать софт делит)

Тезис по тестовому заданию:
Необходимо прописать связи и роли. если это будет апи, то прикрепить постман-коллекцию, если верстка - то разметку. и подготовить миграции и заготовленные сиды, чтобы можно было командой всю бд заполнить и сразу тестить


## Инструкция по запуску

1. Клонировать репозиторий

2. ```composer install```

3. ```cp .env.example .env```

4. Указать в .env свои данные для бд и выполнить миграции: ```php artisan migrate```

5. Заполнить бд данными: ```php artisan db:seed```

6. Запустить встроенный сервер: ```php artisan serve```
